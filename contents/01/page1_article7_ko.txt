##API 키 보안에 대한 안내
고객님의 API 키는 안전합니다

많은 분들이 API 키 보안에 대해 문의해 주시는데,
저희 서비스의 보안 정책을 명확히 안내드리겠습니다.

##🔒 저희 서비스의 보안 원칙

[[purple:API 키 미저장 정책]]
* 고객님의 API 키를 서버에 저장하지 않습니다
* 세션 종료 시 모든 데이터가 초기화됩니다
* 회원가입 없는 서비스로 개인정보 수집 없음.

[[purple: 캐시 미사용 정책]]
* 서버 측 캐시 저장 없음
* 브라우저 세션 기반 일시적 사용만 허용

[[purple: 💡 브라우저 자동완성에 대한 안내 ]]
만약 API 키가 자동완성되거나 저장된 것처럼 보인다면,
이는 당신의 크롬 브라우저의 자동저장 기능 때문입니다.

[[purple: 해결 방법 ]]
1. 브라우저 설정: 자동완성 기능 비활성화
2. 캐시 삭제: 브라우저 캐시 및 쿠키 삭제
3. 시크릿 모드: 개인정보 보호 모드에서 서비스 이용


저희는 강력한 보안 정책을 통해 고객님의 소중한 정보를 보호하고 있습니다.
API 키는 오직 요청 처리에만 사용되며, 그 외의 목적으로는 절대 활용하지 않습니다.

보안은 저희의 최우선 가치입니다.
