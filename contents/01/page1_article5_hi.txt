हे भगवान। मैंने तो बस एक बार ही सर्च किया..?

[IMAGE:05_image_1.png]

बस एक बार सर्च किया!
क्या आपको API कोटा ओवरएज एरर मैसेज दिखा?
मैं वजह बताता हूँ।

[[purple:जितने अधिक देशों में आप सर्च करेंगे, API उतना अधिक कोटा]] खर्च करेगी। 
यदि आपने सर्च में "सभी देश" चुना है,
तो कुल 82 देशों को API रिक्वेस्ट भेजी जाएगी, जिससे काफी कोटा खर्च होता है।

इसलिए, सर्च किए जाने वाले देशों की संख्या कम करें।

---

[IMAGE:05_image_2.png]

## Google Console में कोटा तो बहुत बचा दिख रहा था, फिर भी एरर आ गया।

यह बहुत बार होने वाली समस्या है।
API कोटा उपयोग रियल-टाइम में परिलक्षित नहीं होता,
इसलिए संभव है कि [[purple:Google Console पर दिख रही स्क्रीन में अभी नवीनतम उपयोग परिलक्षित न हुआ हो]]।

---

## सबसे भरोसेमंद समाधान
## नई API कुंजी का उपयोग करना है।

प्रभावी वायरल शॉर्ट्स खोजने के लिए
[[purple: कई API कुंजियाँ पहले से तैयार रखें.]]

[IMAGE:04_image_4.png]

Google असीमित Gmail खाते बनाने की अनुमति देता है,
और प्रत्येक Gmail खाते के लिए अलग-अलग API कुंजी जारी की जा सकती है।
अर्थात, आप [[purple:आवश्यकतानुसार जितनी चाहें उतनी API कुंजियाँ]] सुरक्षित कर सकते हैं।

अभी कई Gmail खातों से API कुंजियाँ जारी करें और नोट में व्यवस्थित कर लें।
