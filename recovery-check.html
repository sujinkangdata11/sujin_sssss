🕰️ 브라우저 백업 확인 가이드

Chrome의 경우:

  1. Chrome 설정 → 고급 → 시스템 → "하드웨어 가속 사용(가능한 경우)" 해제
  2. Chrome://version/ 에서 프로필 경로 확인
  3. 해당 경로의 Local Storage 폴더 확인

수동 확인 경로:

/Users/sujin/Library/Application Support/Google/Chrome/Default/Local Storage/leveldb/

Time Machine 백업이 있다면:

  1. Time Machine 열기
  2. 위 경로로 이동
  3. 어제/최근 백업에서 localStorage 파일 복구
