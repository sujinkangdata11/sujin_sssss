Всего один раз поискали —
и увидели сообщение об ошибке превышения квоты API?
Сейчас объясню почему.

[[purple:Чем больше стран вы выбираете для поиска, тем больше квоты расходует API]]. 
Если при поиске выбраны все страны,
то отправляется запрос API сразу в 82 страны, что сильно расходует квоту.

Поэтому попробуйте уменьшить число стран для поиска.

---

[IMAGE:05_image_2.png]

## В Google Console квоты много, но всё равно возникла ошибка.

Такое происходит очень часто.
Так как расход квоты не отражается в реальном времени,
вполне вероятно, что [[purple:интерфейс Google Console пока не показывает актуальное использование]].

---

## Самый надёжный способ
## — использовать новый ключ API.

Чтобы эффективно находить вирусные шортсы,
[[purple: обязательно подготовьте несколько ключей API заранее.]]

[IMAGE:04_image_4.png]

Google позволяет создавать неограниченное число аккаунтов Gmail,
и для каждого аккаунта Gmail можно получить отдельный ключ API.
То есть вы можете [[purple:обеспечить столько ключей API, сколько потребуется]].

Оформите ключи API в нескольких аккаунтах Gmail прямо сейчас и аккуратно занесите их в заметки.
