Google API? Wat is dat? Voor velen waarschijnlijk nieuw, maar maak je geen zorgen! 

“Moet ik betalen?” → Nee, [[purple:het is volledig gratis.]]
“Heb ik een creditcard nodig?” → [[purple:helemaal niet.]]
“Is het moeilijk?” → [[purple:zelfs een kind van 10 kan het.]]

Google Gemini API is een gratis dienst van Google.

Maak je bijvoorbeeld een `ABC @gmail.com`-account aan, 
dan biedt Google dat account “gratis AI-functies”.
[[purple:Eenvoudig gezegd: het is een van Google’s gratis AI-diensten!]]

[IMAGE:03_image_1.png]


Het is heel eenvoudig – 6 klikken en klaar!
Ik heb zelf een nieuwe API aangevraagd,
en met slechts 6 klikken heb je razendsnel een API-sleutel.
Volg gewoon mee!

##📝 Stapsgewijze gids
Voor wie de video niet kan bekijken, leg ik alles in detail uit met afbeeldingen.
ㅤ
ㅤ
##Stap 1: Ga naar Google AI Studio

Zoek op Google naar [[purple:"ai studio"]].
Klik op de officiële website van Google AI Studio die bovenaan verschijnt.

[IMAGE:03_image_3.png]
ㅤ
ㅤ
##Stap 2: Starten

Klik rechtsboven op “Get started”.

[IMAGE:03_image_4.png]
ㅤ
ㅤ
## Stap 3: Ga akkoord met de voorwaarden

Wanneer het toestemmingsscherm verschijnt, vink het vakje aan en klik op “Confirm”.

[IMAGE:03_image_5.png]
ㅤ
ㅤ
##Stap 4: Start met het aanmaken van een API-sleutel

Klik bovenaan op de blauwe knop “Get API key”.

[IMAGE:03_image_6.png]
ㅤ
ㅤ
##Stap 5: Maak een API-sleutel

Het venster voor het instellen van de API-sleutel verschijnt.
Je hebt uiteraard nog geen sleutel. Klik op “Create new API key”.

[IMAGE:03_image_7.png]
ㅤ
ㅤ
## Stap 6: Klaar!

Klik op “Generate API key” en je bent klaar!
Gefeliciteerd! Je API-sleutel is succesvol aangemaakt.

[IMAGE:03_image_8.png]
ㅤ
ㅤ
## 💡 Handige tips!
[[purple:* Sleutel kwijt? Geen probleem.]]
Met “View API key” kun je hem op elk moment opnieuw bekijken.
[[purple:* Sleutel verwijderen? ]]
Je kunt de bestaande sleutel verwijderen en een nieuwe aanmaken. De nieuwe sleutel is volledig anders en uniek.
ㅤ
Gefeliciteerd!
Via de Google Gemini API kun je nu de geavanceerde AI-functies van Google volop benutten:
tekstgeneratie, beeldanalyse, hulp bij coderen, en meer – gratis!
