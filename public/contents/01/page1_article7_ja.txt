##🔒 当サービスのセキュリティ原則

[[purple: API キー非保存の原則]]
* お客様の API キーはサーバーへ送信・保存しません。
* API キーはブラウザメモリ上でのみ一時的に使用され、ページの再読み込みまたはタブを閉じると削除されます。
* 会員登録／ログイン機能がないため、個人識別情報をサーバーに収集・保管しません。

[[purple: サーバー側保存なし＆ブラウザストレージのご案内]]
* サーバー側のデータベース／セッション／サーバーキャッシュは使用しません。
* サービス利用の利便性のための非機微情報（例：言語設定、表示状態 等）のみをブラウザのローカルストレージに保存し、ブラウザ側で削除されるまで保持されます。
* API キーなどの機微情報は、ローカル／セッションストレージおよびクッキーには保存しません

[[purple: 💡 ブラウザの自動入力に関するご案内]]
API キーが自動入力されたり保存されたように見える場合がありますが、これはご利用のブラウザの自動入力／パスワードマネージャー設定が原因である可能性があります。

[[purple: 解決方法 ]]
1. ブラウザ設定で自動入力／保存機能を無効化
2. ブラウザストレージ（localStorage／キャッシュ／クッキー）を整理
3. シークレットモード（プライバシーモード）を利用

当社は適切なセキュリティ対策を適用し、お客様の情報を保護するよう努めています。  
API キーはリクエスト処理の目的にのみ使用し、それ以外の目的には使用しないことを原則とします。
