##API キーのセキュリティについて
お客様の API キーは安全です

多くの方から API キーのセキュリティについてお問い合わせをいただくため、
当サービスのセキュリティ方針を明確にご案内します。

##🔒 当サービスのセキュリティ原則

[[purple:API キー非保存ポリシー]]
* お客様の API キーをサーバーに保存しません
* セッション終了時にすべてのデータを初期化します
* 会員登録のないサービスのため、個人情報の収集はありません。

[[purple: キャッシュ不使用ポリシー]]
* サーバー側のキャッシュ保存はありません
* ブラウザセッションに基づく一時的な利用のみ許可

[[purple: 💡 ブラウザの自動入力について ]]
もし API キーが自動入力または保存されたように見える場合は、
お使いの Chrome ブラウザの自動保存機能が原因です。

[[purple: 対処方法 ]]
1. ブラウザ設定：自動入力機能を無効化
2. キャッシュ削除：ブラウザのキャッシュとクッキーを削除
3. シークレットモード：プライバシーモードでサービスを利用

当社は強固なセキュリティ方針により、お客様の大切な情報を保護しています。
API キーはリクエスト処理のためにのみ使用され、その他の目的には決して利用しません。

セキュリティは当社の最優先事項です。
