##¿Qué es YouTube Data API v3?

¿Data API? ¿Qué es eso? Puede que muchos lo oigan por primera vez, ¡pero no te preocupes! 

"¿Tengo que pagar?" → No, [[purple:es totalmente gratis.]]
"¿Necesito tarjeta de crédito?" → [[purple:en absoluto.]]
"¿Es difícil?" → [[purple:hasta un niño de 10 años puede hacerlo.]]

YouTube Data API v3 es un servicio gratuito ofrecido por Google.

Por ejemplo, si creas una cuenta `ABC @gmail.com`, 
Google le otorga a esa cuenta el "permiso para consultar datos de YouTube".
[[purple:En pocas palabras, es uno de los servicios gratuitos que ofrece Google.]]

## 📝 Guía textual paso a paso
Para quienes no puedan ver el video, lo explicaré todo con imágenes en detalle.


## Paso 1: Accede a Google Cloud Console

Busca [[purple:"google console"]] en Google. 
Haz clic en el primer sitio web oficial de Google que aparezca.
[IMAGE:02_image_2.png]
ㅤㅤ
ㅤㅤ
## Paso 2: Regístrate

Cuando aparezca la ventana emergente de registro, marca los términos y continúa.
⚠️ **Aviso**: Puede variar por país, pero a veces te intentarán llevar a una suscripción premium ofreciendo 300 USD de crédito.
En ese momento, pulsa el botón "Atrás". [[purple:Solo usaremos la API gratuita]].

[IMAGE:02_image_3.png]
ㅤㅤ
ㅤㅤ
## Paso 3: Busca YouTube Data API

Una vez completes el registro, escribe [[purple:"youtube"]] en la barra de búsqueda superior.
En los resultados, haz clic en [[purple:"YouTube Data API v3"]] que aparece arriba del todo.
[IMAGE:02_image_4.png]
ㅤㅤ
ㅤㅤ
## Paso 4: Habilita la API

Haz clic en este botón azul.
Irás a la página de administración de YouTube Data API.

[IMAGE:02_image_5.png]
ㅤㅤ
ㅤㅤ
## Paso 5: Crea una clave API

Haz clic en la tercera pestaña y luego en [[purple:"+ botón de más"]].
¿A que es sencillo?

[IMAGE:02_image_6.png]
[IMAGE:02_image_7.png]
ㅤㅤ
ㅤㅤ
ㅤㅤ
## 💡 Consejo útil

Si olvidas tu clave API, no te preocupes. 
Siempre puedes consultarla en Google Cloud Console, 
y si lo necesitas, eliminarla y generar una nueva.
