很多朋友咨询 API 密钥的安全性，
在此明确说明我们的服务安全策略。

##🔒 我们的安全原则

[[purple:API 密钥不存储策略]]
* 我们不会在服务器上存储您的 API 密钥
* 会话结束时清空所有数据
* 无需注册的服务，不收集个人信息。

[[purple: 不使用缓存策略]]
* 服务器端不保存缓存
* 仅允许基于浏览器会话的临时使用

[[purple: 💡 关于浏览器自动填充 ]]
如果看到 API 密钥被自动填充或好像被保存，
这是由于 Chrome 浏览器的自动保存功能所致。

[[purple: 解决方法 ]]
1. 浏览器设置：关闭自动填充功能
2. 清除缓存：删除浏览器缓存和 Cookie
3. 无痕模式：在隐私模式下使用本服务

我们以严格的安全策略保护您的重要信息。
API 密钥仅用于处理请求，绝不会用于其他目的。

安全是我们的首要价值。
