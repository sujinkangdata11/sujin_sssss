たった一度検索しただけ！
API の割り当て量超過エラーメッセージを見ましたか？
その理由をお伝えします。

[[purple:検索する国が多いほど、API はより多くの割り当て量]]を消費します。 
検索時に全ての国を選択した場合、
合計82か国に API リクエストを送ることになり、かなりの割り当て量が消費されます。

したがって、検索する国の数を減らしてください。

---

[IMAGE:05_image_2.png]

## Google コンソールで割り当て量が十分残っているのに、エラーが出ました。

これは非常に頻繁に起こる問題です。
API の割り当て量の使用状況はリアルタイムで反映されないため、
[[purple:Google コンソールの表示が最新の使用量をまだ反映していない]]可能性が高いです。

---

## 最も確実な解決方法
## 新しい API キーを使うことです。

効果的なバイラルショートを見つけるために、
[[purple:必ず複数の API キーを事前に準備]]しておいてください。

[IMAGE:04_image_4.png]

Google は Gmail アカウントを無制限に作成できますし、
各 Gmail アカウントごとに個別の API キーを発行できます。
つまり、[[purple:必要に応じていくらでも多くの API キー]]を確保できるということです。

今すぐ複数の Gmail アカウントで API キーを発行し、メモ帳に整理しておきましょう。
