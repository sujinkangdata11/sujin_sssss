##📝 分步骤指南
无法观看视频的朋友，也可结合图片按步骤完成操作。
ㅤ
ㅤ
##第 1 步：进入 Google AI Studio

在 Google 上搜索 [[purple:"ai studio"]]。
点击最上方显示的 Google AI Studio 官方网站。

[IMAGE:03_image_3.png]
ㅤ
ㅤ
##第 2 步：开始

点击右上角的“开始使用”按钮。

[IMAGE:03_image_4.png]
ㅤ
ㅤ
## 第 3 步：同意条款

出现同意页面时，勾选复选框并点击“确认”。

[IMAGE:03_image_5.png]
ㅤ
ㅤ
##第 4 步：开始创建 API 密钥

点击页面顶部的蓝色“获取 API 密钥”按钮。

[IMAGE:03_image_6.png]
ㅤ
ㅤ
##第 5 步：创建 API 密钥

将出现 API 密钥设置窗口。
此时你还没有密钥。点击“创建新的 API 密钥”。

[IMAGE:03_image_7.png]
ㅤ
ㅤ
## 第 6 步：完成！

点击“生成 API 密钥”即可完成！
恭喜！ API 密钥已成功创建。

[IMAGE:03_image_8.png]
ㅤ
ㅤ
## 💡 实用提示！
[[purple:* 忘记密钥了？别担心。]]
点击“查看 API 密钥”可以随时再次查看。
[[purple:* 想删除密钥？ ]]
可以删除现有密钥并新建一个。新密钥与旧密钥完全不同且唯一。
