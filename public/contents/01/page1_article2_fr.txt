## 📝 Guide texte étape par étape
Pour celles et ceux qui ne peuvent pas regarder la vidéo, j’explique tout en détail avec des images.

## 1ère étape : Accéder à Google Cloud Console

Recherchez [[purple:"google console"]] sur Google. 
Cliquez sur le premier site officiel de Google qui apparaît.
[IMAGE:02_image_2.png]
ㅤㅤ
ㅤㅤ
## 2e étape : Inscription

Lorsque la fenêtre d’inscription apparaît, cochez les conditions puis procédez.
⚠️ **À noter** : selon les pays, on peut vous proposer 300 $ de crédit pour vous inciter à souscrire une offre premium.
Dans ce cas, cliquez sur « Retour ». [[purple:Nous n’utiliserons que l’API gratuite]] !

[IMAGE:02_image_3.png]
ㅤㅤ
ㅤㅤ
## 3e étape : Rechercher YouTube Data API

Une fois inscrit, saisissez [[purple:"youtube"]] dans la barre de recherche en haut.
Dans les résultats, cliquez sur [[purple:"YouTube Data API v3"]] en première position.
[IMAGE:02_image_4.png]
ㅤㅤ
ㅤㅤ
## 4e étape : Activer l’API

Cliquez sur ce bouton bleu.
Vous serez dirigé vers la page de gestion de YouTube Data API.

[IMAGE:02_image_5.png]
ㅤㅤ
ㅤㅤ
## 5e étape : Créer une clé API

Cliquez sur le 3e onglet, puis appuyez sur [[purple:"+ bouton plus"]].
C’est très simple, n’est-ce pas ?

[IMAGE:02_image_6.png]
[IMAGE:02_image_7.png]
ㅤㅤ
ㅤㅤ
ㅤㅤ
## 💡 Astuce utile !

Pas de panique si vous oubliez votre clé API. 
Vous pouvez toujours la consulter dans Google Cloud Console, 
et au besoin la supprimer puis en générer une nouvelle.
