##📝 Guia passo a passo
Para quem não puder assistir ao vídeo, explicarei tudo em detalhes com imagens.
ㅤ
ㅤ
##Etapa 1: Acessar o Google AI Studio

Pesquise [[purple:"ai studio"]] no Google.
Clique no site oficial do Google AI Studio que aparece primeiro.

[IMAGE:03_image_3.png]
ㅤ
ㅤ
##Etapa 2: Começar

Clique no botão “Get started” no canto superior direito.

[IMAGE:03_image_4.png]
ㅤ
ㅤ
## Etapa 3: Concordar com os termos

Quando aparecer a tela de consentimento, marque a caixa e pressione “Confirm”.

[IMAGE:03_image_5.png]
ㅤ
ㅤ
##Etapa 4: Começar a criar a chave de API

Clique no botão azul “Get API key” na parte superior da tela.

[IMAGE:03_image_6.png]
ㅤ
ㅤ
##Etapa 5: Criar a chave de API

A janela de configuração da chave de API aparecerá.
É claro que você ainda não tem uma chave. Clique em “Create new API key”.

[IMAGE:03_image_7.png]
ㅤ
ㅤ
## Etapa 6: Concluído!

Clique em “Generate API key” e pronto!
Parabéns! Sua chave de API foi criada com sucesso.

[IMAGE:03_image_8.png]
ㅤ
ㅤ
## 💡 Dicas úteis!
[[purple:* Perdeu a chave? Sem problema.]]
Você pode clicar em “View API key” para vê-la novamente a qualquer momento.
[[purple:* Quer excluir uma chave? ]]
Você pode apagar a chave existente e criar uma nova. A nova chave será totalmente diferente e exclusiva em relação à anterior.
