データAPI？ 何のことでしょう？ 初めて聞く方も多いはずですが、ご安心ください！ 

「お金はかかりますか？」 → いいえ、[[purple:完全無料です。]]
「クレジットカードは必要ですか？」 → [[purple:まったく必要ありません。]]
「難しいですか？」 → [[purple:10歳の子どもでもできます。]]

YouTube Data API v3は、Googleが提供する無料サービスです。

例えば、`ABC @gmail.com` アカウントを作成すると、 
Googleがそのアカウントに「YouTubeデータを照会できる権限」を付与してくれます。
[[purple:つまり、Googleが提供する無料サービスのひとつです！]]

## 📝 段階別テキストガイド
動画が見られない方のために、画像とともに詳しく説明します。


## 1段階: Google Cloud Console にアクセス

Googleで[[purple:"google console"]]と検索してください。 
最上位に表示されるGoogle公式ウェブサイトをクリックします。
[IMAGE:02_image_2.png]
ㅤㅤ
ㅤㅤ
## 2段階: 登録

登録ポップアップが表示されたら、利用規約にチェックして登録を進めてください。
⚠️ **注意事項**: 国や地域によっては、登録と同時に $300 のクレジットを提供するとして、プレミアム登録を促す場合があります。
そのときは「戻る」ボタンを押してください。[[purple:私たちは無料API]]だけを使う予定です！

[IMAGE:02_image_3.png]
ㅤㅤ
ㅤㅤ
## 3段階: YouTube Data API を検索

登録が完了したら、上部の検索バーに[[purple:"youtube"]]と入力してください。
検索結果の一番上に出てくる[[purple:"YouTube Data API v3"]]をクリックします。
[IMAGE:02_image_4.png]
ㅤㅤ
ㅤㅤ
## 4段階: API を有効化

この青いボタンをクリックしてください。
すると、YouTube Data API の管理画面に移動します。

[IMAGE:02_image_5.png]
ㅤㅤ
ㅤㅤ
## 5段階: APIキーの作成

3番目のタブをクリックし、[[purple:"+ プラスボタン"]]を押してください。
とても簡単ですよね？

[IMAGE:02_image_6.png]
[IMAGE:02_image_7.png]

ㅤㅤ
ㅤㅤ
