##📝 Guide étape par étape
Pour celles et ceux qui ne peuvent pas regarder la vidéo, j’explique tout en détail avec des images.
ㅤ
ㅤ
##1ère étape : Accéder à Google AI Studio

Recherchez [[purple:"ai studio"]] sur Google.
Cliquez sur le site officiel de Google AI Studio qui apparaît en premier.

[IMAGE:03_image_3.png]
ㅤ
ㅤ
##2e étape : Commencer

Cliquez sur le bouton « Get started » en haut à droite.

[IMAGE:03_image_4.png]
ㅤ
ㅤ
## 3e étape : Accepter les conditions

Lorsque l’écran de consentement s’affiche, cochez la case puis cliquez sur « Confirm ».

[IMAGE:03_image_5.png]
ㅤ
ㅤ
##4e étape : Commencer à créer une clé API

Cliquez sur le bouton bleu « Get API key » en haut de l’écran.

[IMAGE:03_image_6.png]
ㅤ
ㅤ
##5e étape : Créer une clé API

La fenêtre de configuration de la clé API s’affiche.
Évidemment, vous n’avez pas encore de clé. Cliquez sur « Create new API key ».

[IMAGE:03_image_7.png]
ㅤ
ㅤ
## 6e étape : Terminé !

Cliquez sur « Generate API key » et c’est fini !
Félicitations ! Votre clé API a été créée avec succès.

[IMAGE:03_image_8.png]
ㅤ
ㅤ
## 💡 Conseils utiles
[[purple:* Vous avez perdu la clé ? Pas d’inquiétude.]]
Vous pouvez cliquer sur « View API key » pour la revoir à tout moment.
[[purple:* Vous voulez supprimer une clé ? ]]
Vous pouvez supprimer la clé existante et en créer une nouvelle. La nouvelle clé sera entièrement différente et unique par rapport à l’ancienne.
