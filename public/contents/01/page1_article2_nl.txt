##Wat is YouTube Data API v3?

Data-API? Wat is dat? Voor velen waarschijnlijk nieuw, maar maak je geen zorgen! 

“Moet ik betalen?” → Nee, [[purple:het is volledig gratis.]]
“Heb ik een creditcard nodig?” → [[purple:helemaal niet.]]
“Is het moeilijk?” → [[purple:zelfs een kind van 10 kan het.]]

YouTube Data API v3 is een gratis dienst van Google.

Maak je bijvoorbeeld een `ABC @gmail.com`-account aan, 
dan geeft Google dat account “toegang om YouTube-gegevens te raadplegen”.
[[purple:Eenvoudig gezegd: het is een van de gratis diensten van Google!]]

## Stap 1: Ga naar Google Cloud Console

Zoek op Google naar [[purple:"google console"]]. 
Klik op de eerste officiële Google-website.
[IMAGE:02_image_2.png]
ㅤㅤ
ㅤㅤ
## Stap 2: Registreren

Wanneer de registratiepop-up verschijnt, vink de voorwaarden aan en ga verder.
⚠️ **Let op**: dit verschilt per land; soms proberen ze je te verleiden tot een premium-abonnement met $300 krediet.
Klik dan op “Terug”. [[purple:Wij gebruiken alleen de gratis API]]!

[IMAGE:02_image_3.png]
ㅤㅤ
ㅤㅤ
## Stap 3: Zoek YouTube Data API

Na het registreren typ je [[purple:"youtube"]] in de zoekbalk bovenaan.
Klik in de resultaten op [[purple:"YouTube Data API v3"]] bovenaan.
[IMAGE:02_image_4.png]
ㅤㅤ
ㅤㅤ
## Stap 4: API inschakelen

Klik op deze blauwe knop.
Je gaat dan naar de beheerpagina van de YouTube Data API.

[IMAGE:02_image_5.png]
ㅤㅤ
ㅤㅤ
## Stap 5: API-sleutel maken

Klik op het derde tabblad en druk op de [[purple:"+ plus-knop"]].
Supereenvoudig, toch?
